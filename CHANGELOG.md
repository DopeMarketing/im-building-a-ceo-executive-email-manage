## [0.1.0] - 2026-03-25

### Added

#### Project Foundation
- Initial project scaffold generated by Project Generator
- Next.js 15 application with App Router configuration
- TypeScript strict mode configuration
- Tailwind CSS styling setup
- ESLint and Prettier code formatting
- Complete environment variable configuration template

#### Database Schema
- Supabase PostgreSQL database setup
- Complete data model with 13 tables:
  - `users` table with role-based authentication (Owner, Researcher)
  - `knowledge_base_documents` table for uploaded company documents
  - `departments` table for company organizational structure
  - `training_materials` table for generated training content
  - `sop_templates` table for standard operating procedure templates
  - `sops` table for generated SOPs
  - `crm_integrations` table for HubSpot integration settings
  - `crm_records` table for synced CRM data
  - `slack_configurations` table for Slack workspace settings
  - `slack_notifications` table for notification history tracking
  - `document_recommendations` table for AI-powered suggestions
  - `export_jobs` table for document export job tracking
  - `ai_api_configurations` table for AI service settings
- Row Level Security (RLS) policies for data protection
- Database migrations and schema versioning

#### Authentication System
- Supabase Auth integration

- Authentication pages: login, signup
